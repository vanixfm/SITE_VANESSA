O python é uma linguagem popular para análise de dados, importante por agregar valor aos dados que as empresas possuem, coletados de diferentes aplicações. Com isso, as empresas conseguem adquirir vantagens competitivas frente ao mercado. “O conhecimento oferecido por este curso é um diferencial importante para a carreira profissional. Vamos ajudar a dominar e entender conceitos básicos da linguagem python para análise de dados. Serão introduzidos conceitos como lógica de programação, junção de dados, limpeza e como utilizar bibliotecas como NumPy, pandas e matplotlib. As alunas obterão habilidades de carregamento, inspeção e consulta de dados através da visualização de dados com a linguagem python, além de obter uma visão geral de como responder perguntas básicas sobre esses dados”, detalha Simone Dominico, doutora em Informática pela Universidade Federal do Paraná e coordenadora pedagógica do programa “ELAS no STEM”.

"O curso de especialização em Data Science & Big Data é uma parceria entre os Departamentos de Estatística e Informática da Universidade Federal do Paraná. O curso é composto de três grandes módulos alocados em três semestres. O módulo I (primeiro semestre) apresenta os Fundamentos para Data Science e Big Data. O módulo II (segundo semestre) apresenta as mais avançadas e modernas tecnologias para o processamento de Big Data, bem como, avançadas técnicas de modelagem estatística, tais como, modelos aditivos generalizados, árvores de classificação/regressão e modelos multivariados. No módulo III (primeiro semestre do segundo ano) você vai aplicar as técnicas desenvolvidas durante o curso para construir um produto baseado em dados do mundo real"
